Os contatos são as pessoas com quem sua empresa se comunica pelo SAN Talk AI. Cada contato está vinculado a uma entidade e pode participar de conversas e campanhas.

## Funcionalidades

### Criação e edição
- Crie contatos manualmente ou importe via CSV
- Edite informações como nome, telefone e dados adicionais
- Contatos são criados automaticamente quando um número desconhecido envia mensagem

### Organização
- **Fixar contatos** — marque contatos importantes para acesso rápido
- **Marcar como spam** — filtre contatos indesejados
- **Busca e filtros** — encontre contatos por nome, número ou outros critérios

### Deduplicação
O sistema identifica e permite mesclar contatos duplicados, evitando que um mesmo cliente tenha múltiplos registros.

### Sincronização
Contatos podem ser sincronizados com a Evolution API para manter a base atualizada com os contatos do WhatsApp.

## Importação de contatos

Para campanhas, você pode importar contatos em massa via upload de arquivo CSV. O arquivo deve conter pelo menos o número de telefone de cada contato.

<Note>
  Ao importar contatos, o sistema verifica se o número já existe na base. Contatos existentes são atualizados, novos são criados.
</Note>
