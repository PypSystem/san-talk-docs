## 1. Acesse o Dashboard

Entre em [talk.saninternet.com](https://talk.saninternet.com) com suas credenciais. Se ainda não tem uma conta, fale com nosso time comercial.

## 2. Configure sua empresa

Após o login, o assistente de configuração inicial vai guiar você pelos passos essenciais:

<Steps>
  <Step title="Conecte um canal">
    Vá em **Canais** e conecte seu WhatsApp escaneando o QR Code ou configurando o WhatsApp Business (WABA). [Saiba mais →](/canais/whatsapp)
  </Step>
  <Step title="Alimente a base de conhecimento">
    Vá em **Base de conhecimento** e adicione documentos, perguntas frequentes ou URLs de sites. Esses conteúdos serão usados pela IA para responder seus clientes. [Saiba mais →](/base-de-conhecimento/visao-geral)
  </Step>
  <Step title="Configure a personalidade da IA">
    Vá em **Personalidade** e defina o tom de voz, regras de compliance e diretrizes para o agente de IA. [Saiba mais →](/ia-e-personalidade/personalidade)
  </Step>
  <Step title="Defina horários de atendimento">
    Vá em **Operações > Horários** e configure os dias e horários em que sua equipe atende. Fora do expediente, o atendente passivo assume. [Saiba mais →](/operacoes/horarios-de-atendimento)
  </Step>
</Steps>

## 3. Teste sua configuração

Com tudo configurado, envie uma mensagem de teste para o número do WhatsApp conectado. O agente de IA deve responder usando a base de conhecimento e a personalidade configurada.

<Tip>
  Use o **Playground** no Dashboard para testar respostas da IA antes de ir ao vivo. Ele permite simular conversas com a personalidade configurada.
</Tip>

## 4. Integre via API (opcional)

Se você deseja integrar o SAN Talk AI a outros sistemas, gere uma API key:

1. Acesse **Configurações > API Keys** no Dashboard
2. Clique em **Criar nova chave**
3. Copie a chave gerada (formato `pyp_live_...`)

<CodeGroup>
```bash cURL
curl -X GET https://api-talk.saninternet.com/v1/api/status \
  -H "Authorization: Bearer pyp_live_SUA_API_KEY"
```

```javascript Node.js
const response = await fetch('https://api-talk.saninternet.com/v1/api/status', {
  headers: {
    'Authorization': 'Bearer pyp_live_SUA_API_KEY'
  }
});
console.log(await response.json());
```

```python Python
import requests

response = requests.get(
    'https://api-talk.saninternet.com/v1/api/status',
    headers={'Authorization': 'Bearer pyp_live_SUA_API_KEY'}
)
print(response.json())
```
</CodeGroup>

<Warning>
  Guarde sua API key em local seguro. Ela não será exibida novamente após a criação.
</Warning>
