Os critérios de roteamento definem como as conversas são distribuídas entre seus atendentes. Você pode configurar regras por prioridade para automatizar a atribuição.

## Como funciona

Quando uma nova conversa chega, o sistema avalia os critérios de roteamento na ordem de prioridade:

1. Verifica as regras configuradas
2. Identifica o atendente ou grupo mais adequado
3. Atribui a conversa automaticamente

## Configuração

No Dashboard, acesse **Operações > Roteamento**:

- **Criar critérios** — defina regras de distribuição
- **Ordenar por prioridade** — critérios de maior prioridade são avaliados primeiro
- **Ativar/desativar** — controle quais critérios estão ativos

## Casos de uso

- **Por especialidade** — direcione conversas sobre suporte técnico para atendentes especializados
- **Por volume** — distribua conversas uniformemente entre atendentes disponíveis
- **Por horário** — roteie para equipes diferentes conforme o turno
