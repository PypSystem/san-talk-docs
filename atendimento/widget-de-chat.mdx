O widget de chat é um componente embutível que você pode adicionar ao seu site para conectar visitantes diretamente ao SAN Talk AI.

## Como funciona

O widget é acessado por uma chave pública única associada à sua entidade. Os visitantes do site podem iniciar conversas sem precisar de WhatsApp.

## Configuração

1. Acesse **Canais > Widget de chat** no Dashboard
2. Ative o widget para a entidade desejada
3. Copie o código de incorporação
4. Cole no HTML do seu site

## Funcionalidades

- **Inbox dedicada** — conversas do widget ficam em uma inbox separada
- **Histórico de mensagens** — mensagens anteriores são preservadas
- **Handoff para humano** — a IA pode transferir a conversa para um atendente humano quando necessário
- **Personalização visual** — adapte cores e textos ao visual do seu site

<Note>
  O widget usa autenticação via chave pública (diferente da API que usa `pyp_live_`). Cada entidade tem sua própria chave.
</Note>
